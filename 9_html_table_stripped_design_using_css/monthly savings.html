Monthly Savings

Is it important to save? Yes. Why?

  • So that after you retire you can have that money.
  • So that you can use it in emergencies.
  • So that you can spend it on your children's expenses.
  • Set one specific goal. Rather than socking away money into a savings account, set specific goals for your savings.

How to save money?

  • Budget for savings.
  • Make saving automatic.
  • Keep separate accounts.
  • Monitor & watch it grow.
